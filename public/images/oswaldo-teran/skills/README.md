# Iconos de Skills

Esta carpeta contiene los iconos oficiales de las tecnologías utilizadas en los skills.

## Estructura de archivos requeridos

Los siguientes archivos deben estar en esta carpeta (`public/images/skills/`):

- `html5.svg` - Icono oficial de HTML5
- `css3.svg` - Icono oficial de CSS3
- `javascript.svg` - Icono oficial de JavaScript
- `angular.svg` - Icono oficial de Angular
- `typescript.svg` - Icono oficial de TypeScript
- `wordpress.svg` - Icono oficial de WordPress
- `hubspot.svg` - Icono oficial de HubSpot
- `git.svg` - Icono oficial de Git
- `figma.svg` - Icono oficial de Figma

## Dónde obtener los iconos oficiales

Puedes descargar los iconos oficiales desde:

- **Devicons**: https://devicon.dev/
- **Simple Icons**: https://simpleicons.org/
- **Iconify**: https://iconify.design/
- **Official websites**: Sitios web oficiales de cada tecnología

## Formato recomendado

- Formato: SVG (preferido) o PNG
- Tamaño recomendado: 24x24px o 32x32px
- Fondo: Transparente
- Estilo: Color original de la marca (no monocromo)

## Nota

Si un icono no está disponible, se mostrará la primera letra del nombre de la tecnología como fallback.
